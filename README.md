# 💰 Personal Finance Tracker

A desktop finance management app built with Python, Tkinter and SQLite to track income, expenses and budgets with visual charts.

## ✨ Features
- Add and delete income & expense transactions
- 9 expense categories and 5 income categories
- Real-time pie chart showing expense breakdown
- Set budget limits per category with automatic overspend alerts
- Color-coded transaction history table (green = income, red = expense)
- Persistent storage using SQLite — data saved between sessions
- Clean dark-themed UI

## 🛠️ Tech Stack
- Python 3.11
- Tkinter (GUI + Canvas charts)
- SQLite3 (Local database)

## 🚀 How to Run

1. Clone the repo
   git clone https://github.com/yourusername/finance-tracker.git

2. No external libraries needed

3. Run the app
   python finance.py
